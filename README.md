# Project

เมื่อชนะอาจจะมีการเพิ่มเลเวลหรือได้รับไอเทมหรือสกิลใหม่ที่มีไว้เพื่อคอยช่วยเหลือให้ผู้เล่นเล่นได้ง่ายขึ้น
